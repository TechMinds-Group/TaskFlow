# TaskFlow

## Como Configurar ?
na duvida siga os passos do dockerfile
```sh
pip install -r requirements.txt
pre-commit install
flask create_database
flask create_permissions
```

## Como Rodar ?
na duvida siga os passos do dockerfile
```sh
flask run
```

## Como Ler o Diagrama ?
abra o site [draw.io](https://app.diagrams.net/) e importe o arquivo `diagrama.xml`
